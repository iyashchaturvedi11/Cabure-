## ✨ Features

- ⚡ **Real-time Fare Comparison** — Get live prices from Uber, Ola, and Meru side by side
- 🗺️ **Interactive Route Mapping** — Visualize your journey on the map
- 📊 **Beautiful Visuals** — Clean charts and intuitive UI for easy price analysis
- 📱 **Responsive Design** — Works perfectly on desktop and mobile
- 🎨 **Modern & Attractive Interface** — Built with clean HTML, CSS & JavaScript



## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Visualization**: Custom charts + Leaflet.js / Google Maps API (for route mapping)
- **Real-time Data**: Integrated with Uber, Ola & Meru APIs
